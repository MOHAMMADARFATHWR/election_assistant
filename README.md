# 🗳️ Election Guide Assistant

An interactive, AI-driven Election Guide Assistant built with React. This application provides citizens with a seamless civic interface to understand the voting process, find polling booths, track election timelines, and discover open election datasets.

## ✨ Features

- **🤖 AI Chatbot**: Interactive conversational assistant to answer your questions about voting eligibility, processes, and more.
- **📅 Election Dashboard**: Real-time tracking of upcoming and recently concluded elections with details on polling dates, results, and days left.
- **🗓️ Election Timeline**: A step-by-step interactive timeline explaining the entire democratic process from voter registration to counting day.
- **✅ Voter Journey**: Detailed guide on eligibility, NVSP registration, and what to expect on polling day.
- **📍 Polling Booth Finder**: A mock tool to locate your nearest polling station using your Voter ID or Pincode.
- **⚡ EVM Explainer**: Visual flashcards explaining how Electronic Voting Machines and VVPATs work securely.
- **📊 Open Election Data**: A curated list of the best open-source GitHub repositories for India Election Data (DataMeet, India Votes Data, in-rolls).
- **🎥 Video Tutorials**: Embedded YouTube guides for a better visual understanding of the voting process.

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/MOHAMMADARFATHWR/election_assistant.git
   cd election_assistant
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
